# RuVim 完全ガイド

**Ruby で書かれた Vim ライクなターミナルエディタ**

---

RuVim は Ruby 標準ライブラリのみで動作する、Vim 互換のモーダルエディタです。
Vim の操作感を受け継ぎつつ、Ruby DSL による設定・拡張、26 言語のシンタックスハイライト、
Rich View モード、Git/GitHub 連携、ストリーム連携など、独自の機能を備えています。

本書は RuVim を初めて使う方から、カスタマイズや Plugin API を活用したい上級者まで、
全 26 章 + 付録で包括的に解説するチュートリアルブックです。

---

| 部 | 内容 |
|----|------|
| **第 I 部** はじめの一歩 | 起動・終了、モード、基本移動、テキスト入力、ファイル操作 |
| **第 II 部** 効率的な編集 | オペレータ、テキストオブジェクト、検索・置換、Visual、マクロ |
| **第 III 部** 複数ファイルとワークフロー | バッファ、ウィンドウ、タブ、quickfix、Git/GitHub 連携 |
| **第 IV 部** カスタマイズと拡張 | オプション、設定ファイル、Plugin API、言語サポート、Rich View、ストリーム |

---

```
$ gem install ruvim
$ ruvim
```
