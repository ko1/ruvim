# RuVim 実装解説

**Ruby でテキストエディタを作る**

---

この本では、Ruby で書かれた Vim 風テキストエディタ **RuVim** の実装を、設計思想からコードの細部まで深く解説する。

対象読者は、エディタのプログラミングに興味がある人、あるいは「面白いプログラムの中身を覗きたい」という好奇心を持つプログラマである。

ターミナル制御、キー入力の解析、状態機械によるモード管理、差分レンダリング、Unicode の文字幅処理、C 拡張による高速化、非同期ストリーム、Sixel 画像表示 — テキストエディタの実装には、ソフトウェアエンジニアリングの多くの側面が凝縮されている。

---

```
$ gem install ruvim
$ ruvim
```
